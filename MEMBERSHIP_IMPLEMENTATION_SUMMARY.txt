IMPLEMENTATION DU FLUX D'ADHESION APUNILI
==========================================

RESUME DU FLUX COMPLET:
=====================

1. INSCRIPTION → Utilisateur devient VISITOR
   - RegisterFragment.kt: role = UserRole.VISITOR
   - AuthenticationService.kt: confirmation du role par defaut = VISITOR

2. ACCES A L'ECRAN ADHESION → Affichage conditionnel
   - MembershipFragment verifie le role et l'etat de demande
   - Si VISITOR sans demande → affiche formulaire vide
   - Si VISITOR avec demande PENDING → affiche "En attente"
   - Si VISITOR avec demande ACCEPTED → affiche "Demande approuvee"
   - Si VISITOR avec demande REJECTED → affiche "Demande rejetee" + bouton soumettre

3. SOUMISSION DEMANDE → Status = PENDING
   - MembershipFragment.setupFormSubmission()
   - Appel MembershipViewModel.submitMembershipRequest()
   - Cree une MembershipRequest dans Supabase avec status PENDING
   - Utilisateur reste VISITOR

4. ADMIN APPROUVE → Status = ACCEPTED, Role = MEMBER
   - AdminMembershipFragment.approveMembership()
   - Appel FirebaseMembershipRepository.approveMembership()
   - Met a jour: MembershipRequest.status = ACCEPTED
   - Met a jour: User.role = MEMBER
   - Cree un audit log

5. UTILISATEUR SE RECONNECTE OU RELOAD FRAGMENT
   - LoginFragment recup role = MEMBER depuis DB
   - MembershipFragment.onResume() recharge le statut
   - MembershipViewModel affiche "Demande approuvee"

6. ADMIN REJETTE (ALTERNATIVE)
   - AdminMembershipFragment.rejectMembership()
   - Met a jour: MembershipRequest.status = REJECTED
   - Role reste VISITOR
   - Utilisateur voit message rejet + commentaire admin
   - Peut soumettre une nouvelle demande

LOGIQUE DE VISIBILITE:
====================

Ecran Adhesion selon le role:

VISITOR sans demande:
  - affiche formulaire complet
  - bouton "Envoyer ma demande d'adhesion"

VISITOR avec demande PENDING:
  - affiche "Demande en attente"
  - affiche message d'attente
  - aucun formulaire
  - aucun bouton

VISITOR avec demande ACCEPTED:
  - affiche "Demande approuvee"
  - affiche message "Vous etes maintenant membre"
  - aucun formulaire
  - aucun bouton

VISITOR avec demande REJECTED:
  - affiche "Demande rejetee"
  - affiche commentaire admin
  - affiche bouton "Soumettre une nouvelle demande"

MEMBER:
  - affiche "Vous etes membre"
  - message de confirmation
  - aucun formulaire
  - aucun bouton

ADMIN:
  - redirection automatique vers AdminMembershipFragment
